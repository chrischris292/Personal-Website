Baseball Sabermetrics Project

The project idea came to me after I saw the movie Moneyball. I had a fantasy baseball draft coming up with some friends and needed a way to rank players based off of statistical analysis that no one else could see. Sabermetrics analytics, created by statisticians and economists such as Bill James, often charge a premium to view on websites such as fan graphs. As a financially constrained college student, I had to find alternatives.

This project utilizes PHP, mySQL, HTML, Javascript, and Sean Lehman's Baseball database to implement statistical analysis in Major League Baseball.

Systems Biology Markup Language Project

During the summer, I was involved in an software developer internship related to Systems Biology. I worked on creating a Javascript library used to simulate a chemical reaction encoded in the Systems Biology Markup Language (SBML). The library engineers a system of first-order ordinary differential equations and utilizes the Dormand-Price Method to evaluate the ODE. This project is open source. There exists programs offered in Matlab and other programming languages on the SBML website, but none in a web-based format.

This project utlizes HTML, Javascript, numeric.js, jquery.js.
